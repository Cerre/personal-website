Sports Commentary Search Engine

NLP RAG Video Processing
Challenge: Create a system that allows searching for specific commentator reactions and insights within sports broadcast videos.
Approach: Built a pipeline that downloads YouTube videos, transcribes content, chunks and embeds text, implements RAG with LLM for contextual understanding, and returns precise YouTube links with timestamps.
Technical Insights: Hybrid retrieval approaches significantly outperform pure keyword or semantic search in specialized domains. Sentiment analysis serves as an effective proxy for identifying highlight-worthy moments.

Enhanced Finger Detection Algorithm

Computer Vision Machine Learning XGBoost
Challenge: Improve accuracy of finger detection in challenging conditions like low lighting and occlusion while maintaining computational efficiency.
Approach: Gathered comprehensive dataset across various conditions, engineered features from multiple sources, trained an XGBoost model, and optimized the system for real-time performance.
Technical Insights: Combining hand-crafted features with learned features significantly outperformed either approach alone. Domain knowledge constraints substantially reduced physically impossible predictions.
